## 💻 실행 방법

#### Client 실행

1. 원격 저장소 복제

```bash
$ git clone https://github.com/seongbiny/soboksobok.git
```

2. 프로젝트 폴더로 이동 후 frontend 폴더로 이동

```bash
$ cd frontend
```

3. 필요한 node_modules 설치

```bash
$ yarn install
```

4. 리액트 앱 실행

```bash
$ yarn start
```
